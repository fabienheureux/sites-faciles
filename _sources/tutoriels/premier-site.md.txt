# Créer votre premier site

## 1. Créer un projet Wagtail

Si vous n'avez pas encore de projet Wagtail :

```bash
pip install wagtail
wagtail start mysite
cd mysite
```

## 2. Installer wagtail-dsfr

```bash
pip install wagtail-dsfr
```

## 3. Configurer votre projet

Suivez les instructions du [guide d'installation](../guide/installation.md) pour configurer votre fichier `settings.py`.

## 4. Appliquer les migrations

```bash
python manage.py migrate
python manage.py createsuperuser
```

## 5. Lancer le serveur

```bash
python manage.py runserver
```

Votre site est maintenant accessible à `http://localhost:8000` et l'administration à `http://localhost:8000/admin/`.
